# SO

- Code provided by professor Felipe Viel to the Operating Systems class

- Only Unix-like systems

## How to run

### Change directory to the source folder

```
cd src/
```

### Compile

```
Make
```

### Run

```
./rr <file name>
```

### Clean

```
make clean
```

## Everything in a command

```
make && ./rr rr-schedule_pri.txt && make clean
```
